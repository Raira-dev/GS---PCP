# Global Solution --Pensamento Computacional em Python (PCP)

Este projeto é um sistema desenvolvido em **Python orientado a objetos** para a Global Solution da FIAP.

O objetivo é criar uma **ferramenta inteligente de orientação de carreiras**. O sistema permite que um usuário autoavalie suas **competências técnicas e comportamentais** (como Lógica, Criatividade, Colaboração e Adaptabilidade) e, com base nesses dados, gere **recomendações personalizadas** de carreiras, trilhas de aprendizado ou áreas de aprimoramento.


## Requisitos Técnicos Atendidos

Os códigos desenvolvidos em **Python** e organizados em módulos e classes (`modelo.py` e `main.py`). Há uso de **Listas, Tuplas e Dicionários** para estruturação e análise dos dados; implementação de classes: `Perfil`, `Competencia` e `Carreira`; aplicação de **funções** e **condicionais** para processar dados e gerar recomendações de aderência e aprimoramento; e interface textual simples (CLI)


## Estrutura de Arquivos e Classes

O projeto está dividido em dois módulos principais para organização e coesão: modelo.py e main.py


## Instruções de Execução

### Pré-requisitos
  * Ter o **Python 3.x** instalado na sua máquina.

### 1\. Clonar o Repositório

Abra o terminal e execute os comandos:

```bash
git clone [INSIRA AQUI O SEU LINK DO REPOSITÓRIO]
cd [NOME DO SEU REPOSITÓRIO]
```

### 2\. Executar a Aplicação

Execute o arquivo principal (`main.py`) para iniciar a interface de usuário:

```bash
python main.py
```

O sistema pedirá que você insira seu nome e, em seguida, atribua uma pontuação de **1 a 5** para cada competência listada.
